# Приложение ToDo list

Практика для закрепления ООП и архитектуры MVP на примере
небольшого приложения ToDo.



## Установка и запуск
Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run start
```

или

```
yarn
yarn start
```
## Сборка

```
npm run build
```

или

```
yarn build
```
